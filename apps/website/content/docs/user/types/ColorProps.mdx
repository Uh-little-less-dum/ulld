---
title: ColorProps
---

Many components accept a color as a boolean property. You can both define new colors and override default colors in your configuration file. These colors are available by default.

<Hint>
  While all of these colors are configurable and accomodate both light and dark modes, many change with the app's theme as well. Try toggling the builtin themes by clicking <ThemeSwitch>here</ThemeSwitch>
</Hint>
