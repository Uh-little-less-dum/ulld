## Interfaces

### LargeProps

**`Alpha`**

A utility component to create a piece of large text.

```tsx
<Large center>This large, centered text.</Large>
```

<Large center>This large, centered text.</Large>

#### Properties

| Property    | Type      | Description                                               |
| ----------- | --------- | --------------------------------------------------------- |
| `children`  | `string`  | **`Alpha`** The text to make small.                       |
| `center?`   | `boolean` | **`Alpha`** Make the text centered.                       |
| `muted?`    | `boolean` | **`Alpha`** Make the text take on a muted appearance.     |
| `noMargin?` | `boolean` | **`Alpha`** Remove the default margin around the content. |
