## Interfaces

### LeadTextProps

Lead text is text that is larger than regular text, but of a more muted color.

#### Example

```tsx
<Lead center>This is some lead text.</Lead>
```

<Lead center>This is some lead text.</Lead>

#### Properties

| Property   | Type      | Description      |
| :--------- | :-------- | :--------------- |
| `children` | `string`  | The lead text.   |
| `center?`  | `boolean` | Center the text. |
