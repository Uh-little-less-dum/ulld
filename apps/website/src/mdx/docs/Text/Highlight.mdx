#### Properties

| Property   | Type        | Description                                           |
| ---------- | ----------- | ----------------------------------------------------- |
| `children` | `ReactNode` | **`Alpha`** The text to be highlighted.               |
| `light?`   | `boolean`   | **`Alpha`** Make the highlight semi-tranparent.       |
| `faint?`   | `boolean`   | **`Alpha`** An alias for light.                       |
| `muted?`   | `boolean`   | **`Alpha`** Make the text take on a muted appearance. |
