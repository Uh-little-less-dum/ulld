## Interfaces

### ErrorMarginProps

Used to indicate the error margin of a nearby calculation.

#### Example

```tsx
<ErrorMargin>0.002%</ErrorMargin>
```

### Output

<ErrorMargin>0.002%</ErrorMargin>

#### Properties

| Property     | Type      | Description                                                           |
| :----------- | :-------- | :-------------------------------------------------------------------- |
| `children`   | `string`  | -                                                                     |
| `errorText?` | `boolean` | Whether error content should be colored with the app's 'error' color. |
