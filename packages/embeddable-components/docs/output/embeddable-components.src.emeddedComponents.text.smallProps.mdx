## Interfaces

### SmallProps

A utility component to create a piece of small, subdued text.

```tsx
<Small center muted>
  This some small, centered and muted text.
</Small>
```

<Small center muted>
  This some small, centered and muted text.
</Small>

#### Properties

| Property    | Type      | Description                                   |
| :---------- | :-------- | :-------------------------------------------- |
| `children`  | `string`  | The text to make small.                       |
| `center?`   | `boolean` | Make the text centered.                       |
| `muted?`    | `boolean` | Make the text take on a muted appearance.     |
| `noMargin?` | `boolean` | Remove the default margin around the content. |
