## Interfaces

### EmbeddedImageProps

**`Alpha`**

The Image component extends the html img element with a bunch of style utilities and the ability to more easily utilize images on your local file system without needing to copy or move them.

#### Example

```tsx
<Image
  url="/assets/images/demos/definitelyMe.jpg"
  lg
  center
  desc="Me in the selfie I took for my dating profile."
/>
```

<Image
  url="/assets/images/demos/definitelyMe.jpg"
  lg
  center
  desc="Me in the selfie I took for my dating profile."
/>

#### Properties

| Property     | Type                 | Description                                                                                                                                  |
| ------------ | -------------------- | -------------------------------------------------------------------------------------------------------------------------------------------- |
| `file?`      | `string`             | **`Alpha`** A [root relative path](/docs/terminology?id=rootRelativePath) to an image file.                                                  |
| `url?`       | `string`             | **`Alpha`** A url to a remote image.                                                                                                         |
| `src?`       | `string`             | **`Alpha`** An alias for url                                                                                                                 |
| `image?`     | `string`             | **`Alpha`** The name of a file described in your config. This adds support for easily accessing sets of images, but is currently in beta.    |
| `left?`      | `boolean`            | **`Alpha`** Float image to the left, and wrap other content around it.                                                                       |
| `center?`    | `boolean`            | **`Alpha`** Center align the image, and do not wrap content around it.                                                                       |
| `right?`     | `boolean`            | **`Alpha`** Float image to the right, and wrap other content around it.                                                                      |
| `inline?`    | `boolean`            | **`Alpha`** Setting for inline, small images.                                                                                                |
| `width?`     | `string` \| `number` | **`Alpha`** CSS compliant width or "full" as an alias for `full` and `fullWidth`                                                             |
| `height?`    | `string` \| `number` | **`Alpha`** CSS compliant height                                                                                                             |
| `block?`     | `boolean`            | **`Alpha`** Display image block style, making the entire width of the note reserved for the image, regardless of the image's width.          |
| `full?`      | `boolean`            | **`Alpha`** Make image full width of note, within certain automatic viewport related size constraints.                                       |
| `fullWidth?` | `boolean`            | **`Alpha`** Alias for full.                                                                                                                  |
| `md?`        | `boolean`            | **`Alpha`** Size convenience property                                                                                                        |
| `sm?`        | `boolean`            | **`Alpha`** Size convenience property                                                                                                        |
| `lg?`        | `boolean`            | **`Alpha`** Size convenience property                                                                                                        |
| `xl?`        | `boolean`            | **`Alpha`** Size convenience property                                                                                                        |
| `small?`     | `boolean`            | **`Alpha`** Size convenience property                                                                                                        |
| `medium?`    | `boolean`            | **`Alpha`** Size convenience property                                                                                                        |
| `large?`     | `boolean`            | **`Alpha`** Size convenience property                                                                                                        |
| `border?`    | `boolean`            | **`Alpha`** Add a minimalist border around image                                                                                             |
| `bordered?`  | `boolean`            | **`Alpha`** Alias for border                                                                                                                 |
| `circle?`    | `boolean`            | **`Alpha`** Round image into a circle without distorting it's content.                                                                       |
| `desc?`      | `ReactNode`          | **`Alpha`** An optional description displayed below the image                                                                                |
| `descLeft?`  | `boolean`            | **`Alpha`** Align description to the left                                                                                                    |
| `descRight?` | `boolean`            | **`Alpha`** Align description to the right                                                                                                   |
| `notFigure?` | `boolean`            | **`Alpha`** Do not include image in figure referencing indices.                                                                              |
| `id?`        | `string`             | **`Alpha`** Unique id of image. Optional, but required for figure referencing using the **FigRef** component.                                |
| `label?`     | `boolean`            | **`Alpha`** Apply a default description showing the index of the figure to add a visual reference for pairing with the **FigRef** component. |
