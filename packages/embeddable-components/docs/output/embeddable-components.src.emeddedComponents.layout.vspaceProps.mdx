## Interfaces

### VSpaceProps

The VSpace component is a simple utility to you know, create vertical space between parts of your note.

#### Properties

| Property | Type                 | Description                                                                                                                     |
| -------- | -------------------- | ------------------------------------------------------------------------------------------------------------------------------- |
| `height` | `string` \| `number` | The height of the vertical space in pixels. The default is 16, usually the height of one letter in normal paragraph sized text. |
