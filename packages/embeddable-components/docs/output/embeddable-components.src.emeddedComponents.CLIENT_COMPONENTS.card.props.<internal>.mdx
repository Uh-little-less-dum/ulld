## Interfaces

### BaseEmbeddableComponentProps\<T>

**`Alpha`**

A set of properties used for many embeddable components. These are primarily used when using the component inside of other layout components like grid.

#### Extends

- `HTMLProps`\<`T`>

#### Type Parameters

| Type Parameter                                                                        |
| ------------------------------------------------------------------------------------- |
| `T` _extends_ [`HTMLElement`](https://developer.mozilla.org/docs/Web/API/HTMLElement) |

#### Properties

| Property        | Type      | Description                                                                       |
| --------------- | --------- | --------------------------------------------------------------------------------- |
| `fitParent?`    | `boolean` | **`Alpha`** Whether component should expand to occupy the entire available space. |
| `fitContainer?` | `boolean` | **`Alpha`** Alias for fitParent                                                   |
