## Interfaces

### TitleProps

**`Alpha`**

A simple title component that supports adding a subtitle with reasonable default styles.

#### Example

```tsx
<Title subtitle="And this is my subtitle" subtitleMuted>
  This is my title
</Title>
```

### Output

<Title subtitle="And this is my subtitle" subtitleMuted>
  This is my title
</Title>

#### Extends

- `HTMLProps`\<[`HTMLDivElement`](https://developer.mozilla.org/docs/Web/API/HTMLDivElement)>

#### Properties

| Property         | Type                                                                                          | Description                                                                               | Overrides            |
| ---------------- | --------------------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------- | -------------------- |
| `children`       | `ReactNode`                                                                                   | **`Alpha`** The main title content                                                        | `HTMLProps.children` |
| `subtitle?`      | `ReactNode`                                                                                   | **`Alpha`** The optional subtitle                                                         | -                    |
| `subtitleMuted?` | `boolean`                                                                                     | **`Alpha`** Whether or not subtitle should have a muted appearance. Defaults to **true**. | -                    |
| `subtitleProps?` | `HTMLProps`\<[`HTMLSpanElement`](https://developer.mozilla.org/docs/Web/API/HTMLSpanElement)> | **`Alpha`** Properties passed directly to the subtitle component.                         | -                    |
